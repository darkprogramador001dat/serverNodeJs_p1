O que é memoria RAM? Memória RAM (Random Access Memory) é a memória de trabalho volátil de um computador, celular ou tablet, utilizada para armazenar temporariamente dados de aplicativos e sistemas em execução, permitindo acesso rápido pelo processador. Ela não salva arquivos permanentemente e apaga tudo ao desligar o dispositivo.

O que é HD? O HD (Hard Disk Drive) é o componente de armazenamento permanente (não volátil) de um computador, onde ficam salvos o sistema operacional, programas e arquivos pessoais, mesmo sem energia. Ele funciona mecanicamente, utilizando discos magnéticos giratórios e uma agulha para ler/gravar dados, sendo mais lento e barato que os SSDs.

O que é Dissipador? Um dissipador de calor é um componente metálico (alumínio ou cobre) que absorve o calor excessivo gerado pelo processador (CPU) ou placa de vídeo (GPU) e o dispersa para o ambiente. Geralmente, possui aletas para aumentar a área de superfície, sendo parte fundamental do cooler ativo

O que é Cooler? Cooler é um sistema de refrigeração para componentes eletrônicos (especialmente o processador/CPU) que combina um dissipador de metal e uma ventoinha (fan) para retirar o calor, evitando superaquecimento e danos ao hardware. Eles são essenciais para manter o desempenho e a vida útil do computador, sendo essenciais para o funcionamento correto de processadores.

O que é Processador? O processador, ou CPU (Unidade Central de Processamento)é o "cérebro" do computador, smartphone ou outro dispositivo eletrônico. Ele interpreta, processa instruções e realiza cálculos lógicos e aritméticos, transformando dados em informações para executar programas, navegadores e jogos.

O que são portas SATA? Portas SATA (Serial Advanced Technology Attachment) são interfaces na placa-mãe usadas para conectar dispositivos de armazenamento, como HDDs (discos rígidos), SSDs e unidades de CD/DVD. Elas substituíram o antigo padrão IDE/PATA, oferecendo velocidades mais rápidas (até 6 Gb/s no SATA III), cabos mais finos e eficientes

O que é cabo Flat? O cabo flat (ou flat cable) é um tipo de conector eletrônico plano, fino e flexível, composto por vários condutores metálicos dispostos paralelamente, muito utilizado em eletrônicos como notebooks, impressoras e celulares. Sua principal função é transmitir sinais entre componentes internos, como telas, teclados e placas-mãe, ocupando mínimo espaço.

O que é Sistema Operacional? Um sistema operacional (SO) é o software fundamental que gerencia o hardware (CPU, memória, armazenamento) e fornece uma interface entre o usuário e o computador. Ele carrega programas, organiza arquivos, controla periféricos e gerencia recursos, permitindo o funcionamento de aplicativos. Exemplos: Windows, Linux, macOS, Android e iOS.
